The EnumProcs function takes a pointer to a callback function that will be called once per process with the process filename and process ID.

lpProc – Address of callback routine.

lParam – A user-defined LPARAM value to be passed to the callback routine.

Callback function definition: BOOL CALLBACK Proc(DWORD dw, WORD w, LPCSTR lpstr, LPARAM lParam);
